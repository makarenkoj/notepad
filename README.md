***notepad***
=============

Блокнот в который можно сохранить:
 
 "текстовую запись"
 
 "ссылку"
 
 "задачу"
 
 Блокнот написан на ruby
 
 Для запуска вам понадобится сделать следующее:
 
 1) скачать программу;
 
 2) с помощью консоли или комндной строки зайти в папку с программой
 
 ```
 c:\username>cd ваш путь к папке с программой 
 ```
 
 3) запустить программу используя ruby
 
 ```
 ваш путь к папке с программой>ruby notepad.rb
 ```
  
 4) используя подсказки сделайте свою запись
